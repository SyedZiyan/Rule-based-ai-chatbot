# Rule-Based AI Chatbot 🤖

A simple chatbot built using Python and rule-based logic.

## Features

- Responds to greetings
- Answers predefined questions
- Uses if-else decision making
- Runs continuously using a loop
- Exits using "bye" or "exit"

## Technologies Used

- Python 3

## How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/rule-based-ai-chatbot.git
```

2. Navigate to the project folder

```bash
cd rule-based-ai-chatbot
```

3. Run the program

```bash
python chatbot.py
```

## Sample Output

```text

Type 'bye' or 'exit' to quit

You: hello
Bot: Hello! How can I help you?

You: bye
Bot: Goodbye! Have a nice day.
```
